# Eagle-pnginfo

eagle-pnginfoをEagle APK Tokeに対応させたFork版です。
Stable Diffusion Web UIを動かしているマシンと、Eagleが別PCで動いているとき、設定は正しくされてるのに、localhost:41595 に送ろうとして失敗して画像が送れないことがあります。
設定画面にEagle API Tokenを入れて画像を送れるように改良してみました。
